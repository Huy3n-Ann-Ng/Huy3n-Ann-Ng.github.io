ROTA & PAYROLL for Cooking Class

View Project

Details Summary

This project provides a fully automated solution for managing a cooking class's workflow. It starts with processing booking invoices, followed by creating and managing the staff rota, and finally generating weekly payroll reports. The system integrates Google Sheets formulas to ensure efficiency and accuracy, reducing manual effort and errors.

Project's Written Functions

        Array & Range Manipulation : Arrayformula, Transpose, Bycol, Unique
        Logical : If, Isblank, Isnumber
        Text : Text, Len, Left, Right, Regexmatch
        Lookup & Reference : Vlookup, Col
        Math : Sumif, ABS
        Date : Month
        Custom : Lambda

This script automates the identification of weekdays for each entry in the dataset. The formula evaluates the dates in column `A` and returns the corresponding weekday numbers. This serves as the foundation for generating schedules and payroll calculations.
